# VALIDATE-Python-Scripts

# Analysis code

This repository contains Python scripts used for data analysis and figure generation for the manuscript:

Validation of a Novel Button-Enabled Electronic Adherence Monitoring Device: An Iterative Approach Combining Technical Performance and User Experience 

## Repository structure

- `scripts/`: Python analysis scripts
- `data/`: example input data
- `requirements.txt`: required Python packages

## Requirements

Python 3.11

Install required packages using:

pip install -r requirements.txt

## Usage

Run the analysis scripts from the scripts/ directory.

## Data availability

Example input data are provided where possible.

Full study data are available from the corresponding author upon reasonable request.
